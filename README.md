# parlox-cc

Source code for **[parlox.cc](https://parlox.cc/)** — the personal
directory of Jonathan Ricardo Proaño Alcívar (@parlox): five fronts,
one calling (technology, theology, education, foundation, local church).

Static site (HTML/CSS/JS, no build step). Deployed on Cloudflare Pages.
